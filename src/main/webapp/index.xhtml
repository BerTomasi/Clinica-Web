Enunciado

O trabalho consiste em desenvolver a camada de modelo de uma aplicação, e implementar uma aplicação web para utilização da camada de modelo criada. Abaixo segue a especificação para cada parte do trabalho:

Camada de modelo:

- Implementar no mínimo 5 classes persistentes utilizando a JPA (Java Persistence API). A camada de modelo também deve utilizar anotações para validação de todos os atributos das classes, e pelo menos uma das classes deve possuir um relacionamento de muitos para um.

Camada WEB:

- Desenvolver a aplicação utilizando Java Server Faces.

- Utilizar templates com facelets para as telas da aplicação.

- Implementar manutenções CRUD para pelos menos 3 classes, sendo uma delas com relacionamentos muitos para um.

- As manutenções devem possuir filtro, ordenação e paginação.

Devem ser criados projetos no GitHub para a camada de modelo e para a camada web, e os endereços dos projetos devem ser informados via moodle;

Mini Mundo

Este trabalho se propõe a apresentar um sistema de registros de atendimentos clínicos e de cadastros de médicos e pacientes de um conjunto de clinicas pertencentes ao hospital ficcional Grey Sloan Memorial Hospital (G+SMH).

A proposta é que os pacientes possam ser atendidos em qualquer cidade que possua uma clínica do G+SMH, assim, no sistema os médicos são os únicos que são relacionados com uma clínica de maneira fixa.

Dessa forma, o sistema possui o cadastro dos pacientes e dos médicos (relacionando esses com suas especialidades e clinicas de atendimento) e, por fim, o registro dos atendimentos. Esses últimos são relacionados com os pacientes e médicos.
